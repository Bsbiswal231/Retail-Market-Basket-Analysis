# Retail Market Basket Analysis
## using Apriori Algorithm

## Project Overview
End-to-end Data Analysis project on 
UK Retail Dataset containing 541,909 
real transactions.

## Tools Used
- Python
- PostgreSQL
- SQL
- Power BI

## Key Findings
- 1,450 Association Rules discovered
- Max Lift value of 78x
- Peak Revenue GBP 553,775 (Nov 2011)
- 124% revenue growth Jan to Nov 2011

## Dataset
Kaggle: https://www.kaggle.com/datasets/vijayuv/onlineretail

## Project Structure
- Data_Cleaning.ipynb
- EDA_Apriori.ipynb
- Retail_Market_Basket_SQL_Analysis.sql
- Data/association_rules.csv
- Data/frequent_items.csv
